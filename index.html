Welcome to Architecture.Studio

        
Design and visualize system architectures with a simple, readable text language — no sign-up required.

      

    

    

      

        
📦

        

          
Define components with type + name

          
Use any of 30+ built-in types. Indentation creates containment — child components live inside their parent.

          
server "My Server"
  vm "Worker VM"
    service "App"
    database "PostgreSQL"

        

      

      

        
⇌

        

          
Connect components with arrows

          
Use -> to define connections. Add an optional type like stores_in, publishes_to, etc.

          
"App" -> "PostgreSQL" : stores_in
"App" -> "Redis"
